# 事象カード（Event Cards）

StreetLog のペルソナとバリューシナリオ作成のための事象カード集

## 作成方針
- ペルソナ: 過去の制作者自身の体験をベース
- ユーザーインタビューは実施せず、実体験から抽出
- 具体的な行動・出来事・感情を1カード1項目で記述
- 後で上位下位関係分析で構造化

## メモ欄（追加検討用）

この事象から派生しそうなテーマ：
- トーナメント情報の認知経路
- 配信を見逃した後の行動
- リアルタイム観戦のための準備・障壁
- 見逃さないための工夫・ニーズ
- アーカイブ視聴
- 結果だけ知りたい時
- 好きなプレイヤーの追跡
- 配信チャンネルの選択
- 解説言語の選択
