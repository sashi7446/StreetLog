# 事象カード（Event Cards）

StreetLog のペルソナとバリューシナリオ作成のための事象カード集

## 作成方針
- ペルソナ: 過去の制作者自身の体験をベース
- ユーザーインタビューは実施せず、実体験から抽出
- 具体的な行動・出来事・感情を1カード1項目で記述
- 後で上位下位関係分析で構造化

## 事象カード一覧

### EC-001
**Twitterでトーナメントの盛り上がりを見て、リアルタイムで観戦できなかったことに後悔した**

### EC-002
**プレイヤーのバックグラウンドがわかるときは観戦が楽しい。知らない人でも解説役が紹介してくれるとへぇーってなる**
（視点: オーバーウォッチ観戦 - あえて詳しくないゲームでペルソナの気持ちに近づく）

### EC-003
**界隈の一体感こそスマブラの競技シーンで好きだったところ。ずっとスマブラ観戦をしていた**
（視点: スマブラ観戦経験）

### EC-004
**選手に「全一〇〇」という肩書きがあって、全一同士の戦いはすごく熱く感じた（全一ソニックvs全一ファルコンなど）**
（視点: スマブラ観戦経験）

### EC-005
**「西の神童vs東の神童」のような対比のある対戦カードを見ると興奮した**
（視点: スマブラ観戦経験）

### EC-006

（視点: スマブラ観戦経験）

## メモ欄（追加検討用）

この事象から派生しそうなテーマ：
- トーナメント情報の認知経路
- 配信を見逃した後の行動
- リアルタイム観戦のための準備・障壁
- 見逃さないための工夫・ニーズ
- アーカイブ視聴
- 結果だけ知りたい時
- 好きなプレイヤーの追跡
- 配信チャンネルの選択
- 解説言語の選択
