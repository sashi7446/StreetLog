# 上位下位関係分析 - 価値観の源流を探る

事象カード（EC-001〜EC-041）から、サポートニーズ（〜したい）を抽出し、価値観の源流を見つける

## 分析方針
- 感情、行動の意図、良かった/困った、欲求、否定形、驚きなど多角的に見る
- 過剰解釈を恐れながらゆっくり進める
- 一つずつではなく全体を俯瞰する
- 遠いカード同士でも同じ価値観の源流を持つかもしれない

## メモ：抽象度について
- サポートニーズは「〜したい」の形で、具体的行動と本質的ニーズの中間の抽象度
- 「界隈の鼓動を感じたい」は抽象度が高すぎて本質的ニーズに近い
- 本質的ニーズは、サポートニーズを生み出す根源的な価値観（次のステップで抽出）
