# 上位下位関係分析 - 価値観の源流を探る

事象カード（EC-001〜EC-041）から、サポートニーズ（〜したい）を抽出し、価値観の源流を見つける

## 分析方針
- 感情、行動の意図、良かった/困った、欲求、否定形、驚きなど多角的に見る
- 過剰解釈を恐れながらゆっくり進める
- 一つずつではなく全体を俯瞰する
- 遠いカード同士でも同じ価値観の源流を持つかもしれない

## 次のステップ：価値観の源流を探る

これらのサポートニーズは、さらに上位の価値観につながる可能性がある。

例えば：
- 「そこにある熱を感じたい」+「みんなと一緒に盛り上がりたい」→ **共同体験の価値？**
- 「選手を知る」+「物語性のある対戦」→ **文脈を理解することでの没入？**
- 「今何が起きているか把握」+「ちょうど良いタイミングで知る」→ **参加機会を逃したくない？**
